# Правило: Использование терминала пользователя

Пользователь указал "Всегда используй не свой терминал, а мой, сделай себе такой навык".

Это означает следующее:
1. **Не используй** встроенный инструмент `run_command` для фонового или самостоятельного выполнения системных команд (если только этого не требует сам пользователь).
2. Выводи нужные команды **прямо в чат**, чтобы пользователь мог скопировать их и выполнить самостоятельно в своем рабочем терминале.
3. Если для выполнения задачи (например, установка пакета, запуск скрипта) нужна командная срока, напиши: "Пожалуйста, выполните следующую команду в вашем терминале:" и предоставь блок с кодом.
4. Дождись, пока пользователь подтвердит успешное выполнение, прежде чем продолжать.
